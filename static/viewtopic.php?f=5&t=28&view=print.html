Проброс портов GTA2 и LogeMeIn Hamachi2 через роутер


Эта инструкция актуальна для тех, у кого есть статический IP, но он теряется на домашнем роутере. Если вы не знаете какой у вас IP - скорее всего он динамический, т.к. статический обычно является включаемой опцией интернет-провайдера и оплачивается отдельно. Настраиваем проброску портов GTA2 (чтобы играть напрямую) и LogMeIn Hamachi2 (чтобы со всеми всегда иметь прямой туннель). В качестве примера разбора взят популярный wi-fi роутер DLink DIR-300. Большинство роутеров марки DLink имеют идентичную настройку. Другие роутеры и их подробную настройку под конкретные игры и программы можно найти на сайте Pordforward.com.

Для начала откроем страницу настроек роутера. Введите в браузере адрес 192.168.0.1 (если вы его не меняли - то он именно такой).



Если вы не устанавливали пароль, по-умолчанию для авторизации достаточно ввести логин admin без пароля.



Откройте раздел ADVANCED / Port Forwarding.



Создаем два правила. Игра использует диапазон TCP/UDP 2300-2400 портов и TCP/UDP порт 47624. Настройте правила как указано на рисунке. IP-адрес при выборе вашего компьютера из списка подставляется автоматически - он нужен чтобы роутер знал - кому отправлять пакеты пришедшие на этот порт.



Для LogMeIn Hamachi2 требуется создать три правила, для портов 12975, 17771 и 32976, сделайте это как указано на рисунке. Настройку портов для первой версии Hamachi можно посмотреть здесь.



Еще раз проверяем введенные настройки, галочки, сохраняем.



Далее делаем проверку портов. Создаем игру, вводим здесь порт 2300. Если получаем ответ "Порт открыт" - значит у вас действительно статический адрес и вы все правильно настроили - никаких преград для игры через роутер уже не будет.
